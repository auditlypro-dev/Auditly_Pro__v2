# Auditly Pro

Auditly Pro is a Shopify app designed to help merchants audit their stores for compliance, performance, and optimization issues.

## 🚀 Features (planned)
- Shopify store scanning and auditing
- Compliance checks (policies, listings, store structure)
- Performance and SEO insights
- Automated improvement recommendations
- Future AI-assisted fixes (with user approval)

## 🧱 Tech Stack
- Node.js (backend)
- Express.js
- Shopify Admin API
- OAuth authentication
- Hosted on Render / Cloudflare (planned)
